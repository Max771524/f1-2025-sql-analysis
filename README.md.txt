# F1 賽事資料分析專案

## 📊 專案簡介

本專案以 F1 賽事資料為基礎，透過 SQL 查詢語言進行多面向的資料探索與分析。資料主要來自比賽結果、排位賽、車手資訊、賽道資訊等九張資料表，並透過 SQL 的 JOIN、GROUP BY、子查詢等技巧，深入挖掘有價值的資訊。

## 📁 資料表說明（共 9 張）

| 資料表名稱               | 說明            |
| ------------------- | ------------- |
| `Drivers`           | 車手基本資料（姓名、國籍） |
| `Constructors`      | 車隊資料          |
| `Circuits`          | 賽道資訊          |
| `Races`             | 各年各場賽事資訊      |
| `Results`           | 正賽結果          |
| `QualifyingResults` | 排位賽結果         |
| `RaceResults`       | 整合後的比賽成績資料    |
| `PitStops`          | 進站紀錄          |
| `LapTimes`          | 單圈成績資料        |

---

### 6. 哪個車隊在比賽表現最強？----------------------->6.sql

### 8. 哪一站比賽的名次變動最大？--------------------->8.sql
